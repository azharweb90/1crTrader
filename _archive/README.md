component loader or by `src/app/app-shell.html`. They're kept here for reference only,
in case something in them turns out to be useful later, but they should **not** be migrated
into the React app.

- `rules.html` — unused draft, superseded by the trading-rules section

If you're looking for the live equivalents, see `src/app/features/`.
